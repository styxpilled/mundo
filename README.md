# mundo

Money Man Mundo Goes Where He Pleases

## INSTALLING

WINDOWS:

```bash
git clone https://github.com/styxpilled/mundo
python -m venv venv
venv/Scripts/activate
pip install -r requirements.txt
```

## USAGE

Add your token to `.env` like `TOKEN=yourtokenhere`

```bash
venv/Scripts/activate
python mundo.py
```
